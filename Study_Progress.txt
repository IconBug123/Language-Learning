rather的用法：
rather than = 而不是
or rather = 更确切地说
would rather = 宁愿
rather like = 有点像

程度：相当 → It’s rather nice.
偏好：宁愿 → I’d rather stay.
转折修正：更确切地说 → or rather…
比较选择：而不是 → rather than…


结构：A consists of B + C + D
A = 整体
of 后面列出组成部分
❗ 注意：consist of 不能用于被动语态（不能说 is consisted of）

have 表示已完成的事情对现在还有影响
had 过去的过去，表示过去时间点的区别

to do: 两个动词不能同时用，表示目的，后续的动作(发现...)
claim to, seem to, appear to
to have done 完成的动作

doing sth：1时间，2原因，3条件，4结果(导致)，5虽然，6伴随
一个动作被“当成一件事”时，用 doing，做主语，宾语，定语
修饰名词的定语

done：作定语（修饰名词），作状语（背景），
being done：动作正在发生（被动）
having been done = 被动动作发生在主句动作之前
使用场景：
	原因
	背景
	前置条件
having been done → 因果 / 背景
being done → 进行中（未完成）
done → 状态信息
👉 这些句子通常是：
原因句
条件句
背景句


非谓语作为定语
to do 表示用来...的(将要)
doing 表示(正在...)
done 表示已完成完成的修饰

四个高频万能动词
| 			结构            | 	 实际意思 	|
| ------------------------- | --------- 	|
| have an effect / impact   | 影响        	|
| make a difference         | 产生影响      	|
| place pressure / emphasis | 施加压力 / 强调|
| cause problems / changes  | 导致        	|
直接翻成：
“导致 / 造成 / 产生 / 带来”
