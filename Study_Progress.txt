rather的用法：
rather than = 而不是
or rather = 更确切地说
would rather = 宁愿
rather like = 有点像

程度：相当 → It’s rather nice.
偏好：宁愿 → I’d rather stay.
转折修正：更确切地说 → or rather…
比较选择：而不是 → rather than…

have 表示已完成的事情对现在还有影响
had 过去的过去，表示过去时间点的区别

to do: 两个动词不能同时用，表示目的，后续的动作(发现...)
claim to, seem to, appear to 直接翻译成 “似乎” 后续动作(补语)
to have done 完成的动作

doing sth：1时间，2原因，3条件，4结果(导致)，5虽然，6伴随
一个动作被“当成一件事”时，用 doing，做主语，宾语，定语
修饰名词的定语

done：作定语（修饰名词），作状语（背景），
being done：动作正在发生（被动）
having been done = 被动动作发生在主句动作之前
使用场景：
	原因
	背景
	前置条件
having been done → 因果 / 背景
being done → 进行中（未完成）
done → 状态信息
👉 这些句子通常是：
原因句
条件句
背景句

2 谓语：动词、be动词、四个高能“导致 / 影响”

3 宾语：谓语后面的“受害者 / 目标”

4 补语：把意思“补完整”的成分
| 类型  | 例子               |
| --- | ---------------- |
| 形容词 | become important |
| 名词  | be a problem     |

5 定语：
| 类型   | 信号                                 |
| ---- | ---------------------------------- |
| 形容词  | big problem                        |

| 定语从句 | people **who work remotely**       |

非谓语作为定语
to do 表示用来...的(将要)
doing 表示(正在...)

6 状语：  说明“什么时候 / 为什么 / 怎么做 / 在什么条件下”
		全部是“背景信息”

② 否定前移 & 否定范围

你以后看到这些，立刻警觉：
not necessarily 不一定
not always  并非总是
not entirely 并不完全
not completely 
not fully 
not uncommon = 并不少见
👉 它们否定的是“程度 / 必然性”，不是“事实本身”

形式上的否定在主句，语义上的否定在从句。
看到这些动词 + not，高度警惕：
think
believe
suppose
expect
consider
assume
suggest（有时）
I / We / Researchers do not + 动词 + that 从句
否定通常作用在 that 从句里

X does not think/believe/suggest that Y  ≈ → X thinks/believes/suggests that NOT Y


Not until + 时间/事件 + did + 主语 + 动词
直到 A 发生，B 才发生   A 是前置条件，B 是真正动作  B一直没发生，直到A……    B是倒装
Not until further evidence was presented did the committee revise its conclusion.


Not only … but also …（强调并列型）
不仅 A，而且 B（B 往往更重要）
Not only did + 主语 + 动词原形, but also + 主语 + 动词    (A 是什么，B 就是什么)
句首 not only 要倒装，A 和 B 形式要对称。
例子
	not only skills, but also experience
	not only designed the system, but also tested it
	not only in theory, but also in practice
	The policy affects not only employees but customers as well.
	❌ 错误：not only design, but also testing（形式不对称）


No sooner had A happened than B happened
A 刚一发生，B 就立刻发生

Hardly had + 主语 + 过去分词, when + 一般过去时
 刚刚……就……（几乎还没……就……） 
放在句首要用到装，不放在句首不需要    主语 + had hardly/no sooner + 过去分词 + when/than + 一般过去时

④ 插入语
插入语 = 作者临时塞进来的补充说明，不参与主句的“谁做了什么”。
它的功能是：
	解释背景
	补充细节
	降低语气
	增加信息密度
	但不决定句子的基本意思

插入语特点：
	可删不影响主干
	常用逗号隔开
	表态度/背景
	定语从句特点
	修饰名词
	常用 that / which / who
	不可随便删

插入语 × 定语从句 × 非谓语（三重打断）
